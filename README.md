# 射的ゲーム

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 射的ゲーム
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* マウスでレティクルを操作し、クリックで射撃
* 的に当たればスコアが加算される
* 時間内でのスコアで競う
