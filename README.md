# 射的ゲーム

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 射的ゲーム

## ゲームの遊び方
* マウスでレティクルを操作し、クリックで射撃
* 的に当たればスコアが加算される
* 時間内でのスコアで競う

## ゲームの実装
### 共通基本機能
* ゲームが開始されると的が表示される
　的をクリックすると、スコアが加算される
